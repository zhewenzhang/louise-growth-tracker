## 📞 技術支持與Q&A

### 常見問題

**Q: WHO 曲線出現無限延伸怎麼辦？**  
A: 檢查圖表數據集是否混合了兩種座標格式。確保所有數據都使用 index 0-24 格式，並設置 `spanGaps: false`。

**Q: LocalStorage 數據丟失？**  
A: 檢查瀏覽器設置是否禁用了本地存儲，或清除了 Cookie/緩存。確保使用正確的鍵名 (`louise_*`)。

**Q: 圖表在手機上縮小太多？**  
A: 調整圖表容器高度或使用響應式 `clamp()` 函數調整字體和圖表大小。

**Q: 如何測試深色模式？**  
A: 在 DevTools > Render > Emulate CSS media feature `prefers-color-scheme` 選擇 `dark`。
